Services

What I can do for you

Web Design

Web design encompasses many different skills and disciplines in the production and maintenance of website. The services I provide for Web Designing include web graphic design (Photopshop, InDesign, Illustrator), interface design, authoring, including standardized code and proprietary software, user experience design, and search engine optimization. I am happy to work solo or in a team covering different aspects of the design process.

Web Development

I can govern all the code that will make your website tick. This sort of work can be split into two categories: Front-End and Back-End. In front-end or client-side I will develop a code responsible for determining how the website will display. In the back-end or server-side I will manage data within the database and serve that data to the front-end to be displayed.

Collaborating in a team

I am flexible to work independently or in a team. This includes communcation between front-end and back-end teams when working on wireframes and functionality of the product. Working hand-in-hand is essential to create a working software product. If the teams can get information that they need from each other in a timely fashion then the project has higher probability of going smoothly.
